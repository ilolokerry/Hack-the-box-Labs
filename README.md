# Hack The Box SOC Analyst Learning Path

##  Project Overview

This repository documents my progress through the Hack The Box SOC Analyst Path, focused on developing practical skills in Security Operations, Digital Forensics, and Threat Hunting through hands-on, real-world simulated attack scenarios.

---

##  Objectives

- Build foundational and advanced SOC analyst skills
- Develop incident detection and response capabilities
- Perform malware analysis and forensic investigations
- Strengthen network security and SIEM analysis skills
- Gain hands-on experience with industry-standard security tools
